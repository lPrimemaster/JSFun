Javascript nonsense stuff you can play with

  • orbits - A simple orbital simulator
    • Ctrl + Click + Drag = Place planets (Bugged)
    • First Slider = Simulation speed
    • Second Slider = Camera POV
    • Mouse scroll = Camera zoom
  • simple pendulum - A simple pendular motion simulator
    • No controls for now, might add later...
    • Changing pendulum length via console (type): pendulum.length = 250; restart();
    • Changing simulation delta time via console (type): dt = 0.1;
  • simple spring - A simple (damped) harmonic oscilator simulator
    • No controls for now, might add later...
    • Graph indication horizontal lines are not dynamic when changing values from console!
    • Changing spring length via console (type): spring.l0 = 250;
    • Changing spring damping via console (type): spring.b = 0.1;
    • Changing spring (m/k) via console (type): spring.km = 1 / 10;
    • Changing simulation delta time via console (type): dt = 0.1;
  • simple projectile motion - Launch a projectile considering an air resistance coefficient
    • Use buttons inside the simulation!
  • discrete (static) electric field visualizer
    • Mouse Scroll = Change charge absolute value
    • Space = Switch charge signal
    • Click = Lock charge location and create a new one ate the mouse position
    • A = Disable / Enable mouse charge field effect (nice for visualization purposes)
    • F = Drop positive free charge at mouse location / Delete previously placed free charge
  • funny keys - A midi keyboard vizualizer
    • TODO : Add sound.
  • static cloth simulation - A 2D cloth simulation
    • Simulates a cloth behaviour with structural stress only.
    • TODO : Add shear and bending spring forces.
    • TODO : Add mouse force add interactivity.
